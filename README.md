## Run Locally

Clone the project

```bash
  git clone https://github.com/Udoy2/tech_summit_react_laravel.git
```

Go to the project directory

```bash
  cd tech_summit_react_laravel
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
